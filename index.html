Мини-конспект

1. Анимация Lottie. В данном случае библиотека Lottie подключена через CDN (файл scripts/lottie.min.js), который взят отсюда

2. Затем здесь была выбрана одна из бесплатных анимаций. Кликнув по нужной анимации открывается окно, где через кнопку Download (справа сверху) будет переход на новую страницу с этой анимацией. Там в правой панели нужно загрузить себе анимацию как JSON.

3. JSON файл положить себе в проект, а затем создать в разметке контейнер с уникальным классом или ID для взаимодействия с этим файлом.

4. Затем в скрипте вызвать анимацию по этому примеру:

let animation = lottie.loadAnimation({
container: document.getElementById('уникальный ID'),
render: 'svg',
loop: true,
path: 'путь к JSON'
})


5. При необходимости задать стили

6. Методы для управления анимацией из JS описаны тут. Разные примеры можно глянуть здесь

P.S. вообще JSON должен предоставлять дизайнер

P.P.S. при желании и упорстве в JSON можно изменить цвета. По параметру Fill можно найти массив [] из трех значений - это RGB формат.
